## Reader narrative
<!-- prose-source: agent plane-h 2026-06-28 -->

The pursuit block captures mode (continue, goal_autopilot, company_autopilot), step counters, capability_class, last_verify, evidence_files, and gates_pending. Autopilot loops increment steps_total until check-pipeline-blocked stops ([A3.1](A3.1-session-autopilot-max-steps-per-session.md), [A3.2](A3.2-goal-autopilot-until-goal-verify-or-hard-block.md)).

`last_verify: passed` is mandatory before advancing past implement tasks when evidence_required is true ([G4](G4-evidence-required.md)). gates_pending holds HLD/DD/feature design waits—continue does not clear them ([A5.2](A5.2-continue-not-approval-self-gate-h1-h3-only.md)). route-tier.py applies tier changes when next_action shifts.

## Behavior / step logic
<!-- timeline-source: agent cli-composer-2.5 2026-06-28 -->

1. At H1 or program start, program-scoper selects `pack_id` per [F2.1](F2.1-company-instantiate-program-scoper-pack-select.md); `state.company.pack_id` and default `active_role` bind roles, pipelines, and verify suites from [F1](F1-index.md).
2. Each pursuit wake resolves `active_role` from the pack per [B5.1](B5.1-active-role-from-template-pack.md) and maps `pipeline_id`, skills, and tool permissions per [B5.2](B5.2-role-to-pipeline-id-skills-tool-permissions.md)—the conductor stays thin while workers swap role context per [B5.4](B5.4-conductor-stays-workers-swap-role-context.md).
3. Multi-role companies rotate ready workstreams per [F2.3](F2.3-company-active-role-rotates-ready-work.md) and hand off via manifest and artifact graph per [F2.4](F2.4-company-conductor-handoffs-manifest-graph.md) and [B5.3](B5.3-handoff-manifest-artifact-graph.md).
4. Cross-pack shared libraries import per [F5.1](F5.1-cross-pack-imports-micro-packs.md) within the template-pack ceiling per [F5.3](F5.3-no-repo-outside-template-packs-ceiling.md)—organization changes flow through pack updates, not silent consumer-repo forks.
5. If pack selection is ambiguous, role mapping conflicts, or work would exceed pack ceiling, Plane F stops at H2 rather than proceeding with mismatched roles, pipelines, or verify suites.
