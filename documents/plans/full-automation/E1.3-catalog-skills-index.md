# E1.3: catalog skills index

**Parent:** [E1-index](E1-index.md) · **Branch E:** Knowledge & composition — catalog, compose-first, staleness. · **Vision §7**

## Reader narrative
<!-- prose-source: agent plane-e 2026-06-28 -->

The skills index enumerates `.cursor/skills/` (and `.agents/skills/`) entries—continue, implement-feature, librarian, verifier—each with phase bindings and tool permissions. Catalog discovery prevents conductors from improvising phase behavior already encoded in skills.

Skills rank below scripts and playbooks in [E2.3](E2.3-compose-rank-script-playbook-skill-facts.md) compose ranking because they still invoke model turns; prefer S0 when a skill step is file-derived. Task cards reference skill ids in Components used ([E2.4](E2.4-compose-plan-task-card-components.md)). Pack authors extend the index via template-packs README ([E1.6](E1.6-catalog-template-packs-readme.md)).

## Behavior / step logic
<!-- timeline-source: agent cursor-agent 2026-06-28 -->

1. At phase routing, the conductor resolves skill bindings from the skills catalog index—`.cursor/skills/` and `.agents/skills/` entries with phase tags and tool permissions—instead of inventing ad hoc phase behavior.
2. [E2.3](E2.3-compose-rank-script-playbook-skill-facts.md) ranks skills below S0 scripts and playbooks; the conductor runs [B1.1](B1.1-s0-deterministic-mandatory-first.md) first and invokes a cataloged skill only when file-derived behavior is insufficient.
3. Task cards list skill ids under Components used per [E2.4](E2.4-compose-plan-task-card-components.md); workers spawn with the skill's bounded contract rather than full-repo scope.
4. Pack authors extend the index via template-pack README per [E1.6](E1.6-catalog-template-packs-readme.md); instantiation merges pack skill entries into the active catalog view for role-bound pursuit.
5. If a skill id is missing from the index, permissions exceed active role policy, or compose skips catalog discovery, routing fails closed at H2 until librarian restores a valid indexed skill reference.
