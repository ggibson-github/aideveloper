E7.5 · Knowledge & composition plane

pack, generator, workflow, templates, and inherit

pack generator workflow templates inherit

Template-packs ship more than roles and prose playbooks: they carry generator workflow templates and domain transistor libraries that a new company inherits on day one. Under F1.9, each pack may include workflows/*.json seed DAGs and transistors/*.json blocks for its industry—game studio mesh/rig/import chains, data platform ingest/model/deploy paths, or software delivery scaffold sequences. E7.1 resolves the goal deliverable type to one of these templates before the composer customizes nodes.

Instantiation copies template references into pursuit state; program-scoper does not silently fork pack files into the consumer repo (F5.3). Cross-pack imports via F5.1 may add supplemental workflow fragments. Generic blocks live in F5.4 _shared and merge with pack overlay at query time. Pack CI validates all templates against workflow-dag and transistor v1 schemas before publish.

See Vision §19 — Transistor & generator workflow model and SEC-15-v2.27 for the release that operationalizes pack inheritance.

How it works

  1. Step 1F1.9 defines pack workflows/ and transistors/ directories in pack schema.
  2. Step 2Company instantiate copies template refs into pursuit state; does not fork silently.
  3. Step 3Cross-pack import via F5.1 for micro-pack workflow fragments.
  4. Step 4Pack verify suites (F1.8) may require end-to-end template dry-run in CI.
  5. Step 5Pack ceiling F5.3 applies to transistor forks as with other artifacts.

Edge cases

  • Operator closes laptop mid-loop — state.json must resume from last good dual-write including active_workflow.
  • Transistor version bump mid-pursuit — E5.4 marks workflow stale; re-validate before next node.
  • L0 waiver node without promotion progress — D3.3 priority boost then H2 if threshold exceeded.
  • Pack overlay id collision — F5.4 semver fork per D5.3, not silent overwrite.
  • Parallel branch join missing typed input — validate-workflow-dag fails at compose time.
