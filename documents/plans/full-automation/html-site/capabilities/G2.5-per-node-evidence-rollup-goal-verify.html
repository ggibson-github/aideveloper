G2.5 · Verification & quality plane

per-node evidence rollup goal verify

per-node evidence rollup goal_verify

Goal-level verification must reflect workflow execution, not only legacy task-card checkboxes. This capability extends G2.2: when state.pursuit.active_workflow is bound, goal-verify.py accepts a manifest of evidence paths collected from each entry in completed_nodes[]—each path produced by that node's transistor verify command or evidence_path_template. Missing node evidence fails rollup even if an implement task was marked done in chat, closing the G5.1 hallucinated-done class at workflow granularity.

Tool and MCP outputs use evidence types from I4.3 when transistors invoke external executors (I4.4). G2.4 regression batches may run a subset of transistor verify suites after each implement batch. Rollup success is prerequisite for G2.3 H3 transition per INTRO-1.3. Immutable logs remain under H4.

See Vision §19 — Transistor & generator workflow model and C6.5 for terminal-node completion rules.

flowchart TD
  compose[Compose workflow] --> validate[validate-workflow-dag S0]

How it works

  1. Step 1goal-verify.py accepts --workflow-evidence manifest from active_workflow.completed_nodes.
  2. Step 2Missing node evidence fails rollup even if legacy task cards marked done.
  3. Step 3Tool/MCP evidence types per I4.3 included in manifest when transistors use tool executors.
  4. Step 4Regression batch G2.4 runs transistor verify suite subset on implement batches.
  5. Step 5Evidence immutability H4 applies per node path.

Edge cases

  • Operator closes laptop mid-loop — state.json must resume from last good dual-write including active_workflow.
  • Transistor version bump mid-pursuit — E5.4 marks workflow stale; re-validate before next node.
  • L0 waiver node without promotion progress — D3.3 priority boost then H2 if threshold exceeded.
  • Pack overlay id collision — F5.4 semver fork per D5.3, not silent overwrite.
  • Parallel branch join missing typed input — validate-workflow-dag fails at compose time.
