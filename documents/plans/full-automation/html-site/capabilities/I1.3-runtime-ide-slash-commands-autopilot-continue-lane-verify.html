runtime IDE slash commands autopilot continue lane verify

Slash commands—/continue, /autopilot, /status, /gate, /task, /verify, /lane, /program—map to skills in .cursor/skills/ and inject deterministic behavior before the LLM turn. Commands are the operator's control surface; they do not substitute for H2 answers or H3 approval.

/continue performs exactly one pipeline step; /autopilot loops until check-pipeline-blocked (A3). Command definitions live under .cursor/commands/ and must stay aligned with skill contracts. SDK and headless runners invoke equivalent script entry points (I2.1).
