A2.7 · Pursuit & control plane

no, intermediate, wait, for, and continue

no intermediate wait for continue

Earlier automation models asked humans to type “continue” after every step. This capability forbids intermediate wait-for-continue: the loop runs autonomously between H1, H2, and H3 touchpoints.

“Continue” in operator language now means one pipeline step when manually invoked—not a prompt between every phase inside autopilot. That redefinition is what makes 100% automation between touchpoints achievable.

Confusion here causes products to re-introduce manual babysitting. This spec is the guardrail against regressing to burst-and-wait workflows.

How it works

  1. Step 1With autopilot.active true, the conductor runs the A2.6 canonical loop—A2.1 preflight, one A2.2 phase, A2.3 post-step—back-to-back in the same session without prompting the operator to type "continue" between skill phases.
  2. Step 2Manual /continue or operator "continue" executes exactly one pipeline step only when autopilot is inactive; it does not babysit each phase inside an active pursuit loop.
  3. Step 3Permitted human pauses are H1 plan approval, H2 blocker resolution, and H3 final sign-off.; check-pipeline-blocked BLOCKED exits the loop until H2 clears.
  4. Step 4Skills, rules, and run-local-pipeline must not reintroduce intermediate wait-for-user gates between post-step dual-write and the next preflight except at H1, H2, or H3 touchpoints.
  5. Step 5If a conductor turn ends while autopilot remains active without advancing pursuit or dual-writing an H2 stop reason, the next wake fails closed at preflight BLOCKED until journal and state.json reconcile the stall.

Edge cases

  • Operator closes laptop mid-loop — state.json must resume from last good dual-write.
  • Concurrent manual edit to queue JSON — conductor reloads queue each wake; last writer wins with journal note.
  • goal_verify passes but H3 rejected — goal.state returns to pursuing with rejection notes.
  • Edge case A2.7 variant 4: verify state dual-write before continuing pursuit.
