runtime SDK operator PC worker server

Operator PC as worker server extends SDK daemon: a dedicated machine drains work-order lanes, runs verify suites, or hosts GPU/tool workloads while the conductor IDE session stays thin. Lease semantics follow C4.1 lane JSON.

Network partition between conductor and worker server triggers H2 with last-known lease state—workers must not assume conductor is reachable for journal writes. Evidence and logs write to shared evidence/ paths or sync back on completion.
