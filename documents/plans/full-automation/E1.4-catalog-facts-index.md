# E1.4: catalog facts INDEX

**Parent:** [E1-index](E1-index.md) · **Branch E:** Knowledge & composition — catalog, compose-first, staleness. · **Vision §7**

## Reader narrative
<!-- prose-source: agent plane-e 2026-06-28 -->

`docs/facts/INDEX.md` is the entry point for external truth: URLs, credentials locations, API endpoints, team contacts, and captured snippets. Agents must read INDEX before guessing configuration—the facts-retrieval rule and [E3.1](E3.1-facts-docs-facts-external-truth.md) enforce this.

Facts rank lowest in compose preference ([E2.3](E2.3-compose-rank-script-playbook-skill-facts.md)) because they inform but do not execute. The remember skill ([E3.3](E3.3-remember-skill-index-no-fts-global.md)) appends to captured facts and updates INDEX—no global FTS memory. Librarian may include one fact path in allowed_reads ([E4.3](E4.3-context-allowed-reads-cap-max-5.md)).

## Behavior / step logic
<!-- timeline-source: agent cursor-agent 2026-06-28 -->

1. Before guessing URLs, credentials, API endpoints, or team contacts, the conductor and workers read `docs/facts/INDEX.md` per facts-retrieval and [E3.1](E3.1-facts-docs-facts-external-truth.md)—external truth is never invented in-model.
2. Librarian may include at most one fact file path in allowed_reads per [E4.3](E4.3-context-allowed-reads-cap-max-5.md); pursuit loads the indexed fact snippet before routing workers to integration or deploy steps.
3. [E2.3](E2.3-compose-rank-script-playbook-skill-facts.md) ranks facts lowest in compose preference—facts inform decisions but do not execute; scripts and playbooks still run first when available.
4. When operators capture new truth via remember ([E3.3](E3.3-remember-skill-index-no-fts-global.md)), the skill appends to captured facts and updates INDEX—there is no global FTS memory outside the catalog.
5. If INDEX is missing an entry for a required configuration key, fact files conflict, or workers proceed without facts lookup on high-risk external calls, pursuit stops at H2 until dual-write records the gap and operator supplies indexed truth.
