# A4.3: stop resource max_steps max_cost lease_expired

**Parent:** [A4-index](A4-index.md) · **Branch A** · **Vision §3** · **Release:** v2.15

## Reader narrative
<!-- prose-source: agent plane-a 2026-06-28 -->

Resource stops bound autonomous pursuit in measurable units: max steps per session or goal, token budgets, cost caps, and lease expiry on shared runners. When a budget is exhausted, pursuit halts with a stop reason that operators can extend at H2 if policy allows.

These limits implement [A1.4](A1.4-deadline-budget-steps-tokens-wall-clock.md) in the stop taxonomy so “always-on” never means “unbounded.” Scheduling in Plane D may reprioritize work when leases expire, but the stop record must remain auditable. Too-tight budgets cause false stops; too-loose budgets waste spend—pack authors set defaults per goal_type.

## Behavior / step logic
<!-- timeline-source: agent cursor-agent 2026-06-28 -->

1. Each pursuit wake increments budget counters from [A1.4](A1.4-deadline-budget-steps-tokens-wall-clock.md)—session steps, goal steps, token spend, cost accrual, and runner lease timestamps—before [A2.2](A2.2-if-ready-execute-one-pipeline-step.md) schedules another phase.
2. When any configured ceiling is reached, [A2.1](A2.1-preflight-check-pipeline-blocked-extended.md) emits stop class `resource:max_steps`, `resource:max_cost`, or `resource:lease_expired` and exits [A2.6](A2.6-loop-until-blocked-budget-achieved-h3-reject.md) without advancing `next_action`.
3. Journal-keeper dual-writes the exhausted budget dimension, current counter values, and stop reason to journal and state.json so operators and Plane D schedulers can audit why pursuit halted—not a silent session end.
4. Operators may extend budgets at H2 when policy allows; [D3.2](D3.2-priority-boost-queue-depth-threshold.md) may reprioritize sibling work when a shared runner lease expires, but the stop record remains until explicitly cleared.
5. If budget fields are missing or counters disagree with ledger evidence, preflight fails closed at H2 rather than allowing unbounded always-on pursuit.
